Enter command in format

Console.NumToLCD #
(example: Console.NumToLCD 1234)

or

Console.NumToLCD -s # #
(example: Console.NumToLCD -s 3 56)

-s is an optional control that changes the size of the LCD numbers (default size is 2).


If LCD.Change file is present, change file name back to LCD.exe. File name changed in order send through email.
